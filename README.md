# GDevelop-Batch-GLB-Import
CLI script for batch GLB Import into GDevelop.

1. Install Node.js
2. Place the Script into Your Project folder.
3. Change the Project line to match your JSON. (Line 6.)
4. Place all Models in assets/models directory of your project. (Heirarchy from there can be whatever.)
5. Run Script.
6. All objects will be global and separated into folders in project to match your models folder.
